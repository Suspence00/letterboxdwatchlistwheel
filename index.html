        ?

        How to get your Radarr API key:

        1. Open Radarr and go to Settings → General.
        2. Find the API Key under the Security section.
        3. Copy the key and paste it below.

        Note: If Radarr is behind a reverse proxy, you may need to enable CORS headers.
